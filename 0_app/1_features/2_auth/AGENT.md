# Auth Feature Rules

- Use this folder only for the `2_auth` feature.
- Keep proposal files next to the feature:
  - `spec.md`
  - `acceptance.md`
  - `feature.manifest.yaml`
  - `AGENT.md`
- Do not implement code until the feature is approved.
- After approval, keep changes inside `0_app/1_features/2_auth/` unless a shared concern is explicitly approved.
